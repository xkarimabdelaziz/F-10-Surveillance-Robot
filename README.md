# F-10

F-10 is a smart wireless surveillance robot designed for warehouses and industrial environments.

## Features

- Wireless control using ESP32 and nRF24L01
- Obstacle detection using Ultrasonic Sensor
- Fire detection using Flame Sensor
- Rain / water detection
- Motion detection using PIR Sensor
- Servo-based environmental scanning
- Smart buzzer alert system

## Components Used

- Arduino Nano
- ESP32
- nRF24L01 Modules
- L298N Motor Driver
- Ultrasonic Sensor (HC-SR04)
- Flame Sensor
- Rain Sensor
- PIR Motion Sensor
- Servo Motor
- Buzzer
- LED
- DC Motors

## Project Idea

The robot is designed to move wirelessly and monitor the environment in real time.  
It can detect obstacles, fire, motion, and water, then react using alerts and warning systems.
